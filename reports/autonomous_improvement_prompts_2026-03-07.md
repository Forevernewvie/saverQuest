# SaverQuest Autonomous Improvement Prompts

## Prompt 1
현재 Flutter 앱의 공통 레이아웃을 서비스형 UI로 정리해줘.
조건:
- 모든 화면에서 같은 디자인 언어가 느껴져야 함
- AppBar, 배경, 스크롤 패딩, 화면 폭 제한을 공통 규칙으로 통합
- 과한 장식은 피하되 프로토타입 느낌은 제거
출력:
1) 수정할 공통 파일 목록
2) 변경 이유
3) 적용 후 검증 명령

## Prompt 2
광고 미설정 또는 동의 미완료 상태가 사용자에게 공격적으로 보이지 않도록 다듬어줘.
조건:
- 광고 placeholder 문구를 최소화
- 동의 미완료 시 사용자에게 기술 상태를 드러내지 않음
- 광고 슬롯은 화면 흐름을 깨지 않아야 함
출력:
1) 수정 정책
2) 수정 파일
3) QA 체크포인트

## Prompt 3
리포트 화면을 현재 수익화 설정에 맞게 현실적으로 조정해줘.
조건:
- 현재는 배너만 운영 중
- 리워드 광고 단위 ID는 아직 없음
- 없는 기능을 있는 것처럼 보이게 하면 안 됨
출력:
1) 사용자에게 보일 상태 정의
2) 화면별 fallback 문구
3) 테스트 케이스

## Prompt 4
이번 개선 내용을 다시 실행 가능하도록 문서화해줘.
조건:
- 무엇을 왜 바꿨는지 바로 이해 가능해야 함
- 검증 명령과 성공 기준 포함
출력:
1) 실행 결과 요약
2) 남은 후속 작업
